Como correr um programa java?
1 - compilar o programa:
 -> javac programa.java
2 - como correr o programa:
 -> java programa [argumentos ]

Como correr um programa python?
1 - Apenas correr o programa:
 -> python programa.py [argumentos]

Comandos do Utilizador disponiveis:
1 - User:
 -> list (lista as linguagens disponiveis)
 -> 	request [numero da linguagem que se quer] [t (palavras)] [n Palavras] 
	ou 
	request [numero da linguagem que se quer] [f (file)] [nome ficheiro] 
 -> exit (acaba o programa)

Quantas traducoes diferentes existem?
Existem 4 traducoes de palavras (Ingles, Frances, Holandes , Dinamarques)
Existe 1 traducao de ficheiros de imagem (Ingles)

Como fazer kill ao TRS ? 
A cada comunicacao com um user, o TRS pergunta na linha de comandos
se o utilizador que o controla se quer ou nao terminar o programa.

Como fazer kill ao TCS ?
Necessita apenas de fazer um ctrl+c.
